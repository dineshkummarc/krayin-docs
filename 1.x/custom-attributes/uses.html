# Create and using a new attribute

Here is the list of all attributes

  • Step-1: Navigate to Settings menu and search about Attributes click on it.

  • Step-2: Now click on Create Attribute button.

  • Step-3: Fill all the inputs field as your requirement the hit the Save as Attribute button.

  • Step-4: Your custom attribute is created now and visible into Leads create page when you create Leads.
